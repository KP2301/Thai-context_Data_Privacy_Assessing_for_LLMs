# dea_result

โฟลเดอร์นี้เก็บผลลัพธ์ที่ได้จากการรัน Data Extraction Attacks (DEA)

## โครงสร้างหลัก
- `without_defence/` - ผลลัพธ์จากการโจมตีโดยไม่ใช้การป้องกัน
- `defence/` - ผลลัพธ์จากการโจมตีเมื่อใช้ defensive prompts หรือการล้างคำ

## การใช้งาน
- หลังจากรัน `dea_with_metric.py` หรือ `dea_with_dp.py` ผลลัพธ์จะถูกบันทึกไปยัง `output/dea/` และคัดย้ายไปยังโฟลเดอร์นี้เพื่อเก็บถาวร
- หากต้องการวิเคราะห์เพิ่มเติม ให้ใช้สคริปต์ในโฟลเดอร์ `analyze/dea/` หรือ `analyze/gather`

## หมายเหตุ
- โครงสร้างโฟลเดอร์ช่วยให้แยกผลลัพธ์ก่อน/หลังป้องกันได้ชัดเจน
