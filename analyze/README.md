# analyze

โฟลเดอร์นี้เก็บสคริปต์สำหรับวิเคราะห์ผลลัพธ์จากการโจมตี (DEA/JA) และสร้างกราฟหรือรายงานสรุปข้อมูล

---

## โครงสร้างหลัก

- `dea/` - การวิเคราะห์ผลลัพธ์จาก Data Extraction Attacks (DEA)
  - `language/` - วิเคราะห์ตามภาษา (ไทย/อังกฤษ) เพื่อดูการแทรกแซงที่แตกต่างกัน
  - `length/` - วิเคราะห์ตามความยาวของพรอมต์หรือข้อความ เพื่อดูผลกระทบของ context size
  - `token/` - วิเคราะห์ตามจำนวนโทเคนเพื่อดูผลต่อ tokenization และการรั่วไหล

- `ja/` - การวิเคราะห์ผลลัพธ์จาก Jailbreak Attacks (JA)
  - `jailbreak_rate.py` - สร้างกราฟเปรียบเทียบ success rate ระหว่างโมเดลและระดับภาษา
  - `ja_analyze_all_result.py` - สรุปค่าเฉลี่ยจากไฟล์ผลลัพธ์ (ทั้งไทยและอังกฤษ) และสร้างกราฟแบบ clustered bar

- `gather/` - สคริปต์สำหรับรวบรวมผลลัพธ์จากไฟล์หลายรุ่น เช่น ผลลัพธ์จากหลายโมเดลหรือหลายรอบ
  - `gather_3_models.py` - รวมไฟล์ผลลัพธ์จากโมเดลหลายตัวเป็นไฟล์เดียว และคำนวณ ASR รวม

- `picture/` - เก็บรูปภาพหรือกราฟที่สร้างขึ้นจากการวิเคราะห์ (ผลลัพธ์ PNG/ภาพต่าง ๆ)

---

## การใช้งานโดยรวม

1. **รันโจมตีเพื่อสร้างผลลัพธ์**
   - DEA: `python dea_with_metric.py` หรือ `python dea_with_dp.py`
   - JA: `python ja_with_metric.py` หรือ `python ja_with_dp.py`

2. **เตรียมไฟล์ผลลัพธ์**
   - ผลลัพธ์จะถูกเขียนลงใน `output/dea/` หรือ `output/ja/` ตามประเภทการโจมตี
   - หากต้องการเก็บสำรอง ให้ย้ายไฟล์ไปยัง `dea_result/` หรือ `ja_result/`

3. **วิเคราะห์ผลลัพธ์**
   - เรียกใช้งานสคริปต์ในโฟลเดอร์ `analyze/` เช่น `python analyze/ja/jailbreak_rate.py` หรือสคริปต์อื่น ๆ ในโฟลเดอร์ย่อย
   - สคริปต์เหล่านี้มักจะอ่านไฟล์ `.txt` ที่มี `Rate : XX.XX%` แล้วสร้างกราฟหรือสรุปเป็นตัวเลข

4. **ดูผลลัพธ์ที่สร้างขึ้น**
   - ตรวจสอบโฟลเดอร์ `analyze/picture/` หรือไฟล์ที่สคริปต์ระบุไว้ (เช่น `thai_jailbreak_benchmark.png`, `jailbreak_rate_clustered.png`)

---

## ตัวอย่างการรันสคริปต์การวิเคราะห์

### 1) รันกราฟ Jailbreak Rate

จากโฟลเดอร์หลักของโปรเจกต์:

```bash
python analyze/ja/jailbreak_rate.py
```

- จะค้นหาไฟล์ `ja_output*.txt` ในโฟลเดอร์ปัจจุบัน
- สร้างกราฟเปรียบเทียบ success rate ระหว่างโมเดลสำหรับแต่ละระดับภาษา
- บันทึกไฟล์ `thai_jailbreak_benchmark.png`

### 2) สรุปผล Jailbreak จากหลายรอบ

```bash
python analyze/ja/ja_analyze_all_result.py
```

- จะอ่านไฟล์ผลลัพธ์จากไดเรกทอรีที่กำหนด (`base_th` และ `base_eng`) และคำนวณค่าเฉลี่ย
- สร้างภาพกราฟ clustered bar พร้อมสรุปตัวเลขค่าเฉลี่ย

### 3) รวมผลลัพธ์ในโฟลเดอร์เดียว

```bash
python analyze/gather/gather_3_models.py
```

- สร้างไฟล์รวม `FINAL_TOTAL_REPORT.txt` จากไฟล์ `.txt` หลายไฟล์
- เหมาะสำหรับการรวมผลลัพธ์จากโมเดลหลายตัวหรือหลายรอบ

---

## แนวทางการเพิ่มการวิเคราะห์ใหม่

1. เพิ่มสคริปต์ Python ใหม่ในโฟลเดอร์ที่เหมาะสม (เช่น `analyze/dea/` หรือ `analyze/ja/`)
2. อธิบายวิธีใช้งานในส่วนหัวของสคริปต์ (docstring) และเพิ่มตัวอย่างคำสั่งที่ใช้ใน README นี้
3. หากสคริปต์สร้างไฟล์กราฟหรือรายงาน ให้เก็บไว้ใน `analyze/picture/` เพื่อความเป็นระเบียบ

---

## หมายเหตุเพิ่มเติม

- โฟลเดอร์ `analyze/` ไม่ควรแก้ไขผลลัพธ์ดิบโดยตรง แต่ควรอ่านจากไฟล์ผลลัพธ์ที่สร้างจากสคริปต์โจมตี
- หากโครงสร้างไฟล์ผลลัพธ์เปลี่ยนแปลง (เช่นฟอร์แมต `Rate : XX.XX%`), ควรปรับฟังก์ชัน parsing ในสคริปต์ที่เกี่ยวข้อง (เช่น `parse_jailbreak_file`)
