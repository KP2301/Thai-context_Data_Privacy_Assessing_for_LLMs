# data

โฟลเดอร์นี้เก็บข้อมูลที่ใช้สำหรับการทดสอบการโจมตี (DEA และ JA)

## โครงสร้างหลัก
- `enron/` - ข้อมูล dataset Enron สำหรับการโจมตี Data Extraction (DEA)
- `advbench/` - ข้อมูลชุดคำถามจาก AdvBench สำหรับการโจมตี Jailbreak (JA)
- `jailbreakqueries.py`, `jailbreakqueries_th.py`, `jailbreak_th_data.py` - สคริปต์สำหรับสร้าง/โหลดชุดโจทย์ Jailbreak

## การใช้งาน
- ใส่ข้อมูลทดสอบลงในโฟลเดอร์ที่เกี่ยวข้อง
- หากต้องการเพิ่มชุดข้อมูลใหม่ ให้สร้างไฟล์หรือโฟลเดอร์ใหม่และอัปเดตโค้ดที่โหลดข้อมูล

## หมายเหตุ
- อย่าเก็บ API keys หรือข้อมูลความลับในโฟลเดอร์นี้ (ใช้ `.env` สำหรับคีย์)
- ไฟล์ข้อมูลอาจมีขนาดใหญ่ ควรจัดการพื้นที่เก็บข้อมูลให้เพียงพอ
