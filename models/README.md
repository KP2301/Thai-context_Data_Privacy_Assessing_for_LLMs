# models

โฟลเดอร์นี้ประกอบด้วยโมดูลสำหรับเชื่อมต่อและใช้งานกับผู้ให้บริการ LLM ต่าง ๆ

## โครงสร้างหลัก
- `LLMBase.py` - คลาสฐาน (base class) สำหรับโมเดลทุกตัว
- `GroqModels.py` / `GroqModels_dp_version.py` - การเชื่อมต่อกับ Groq API
- `OpenAI.py` - การเชื่อมต่อกับ OpenAI API
- `TogetherAI.py` - การเชื่อมต่อกับ TogetherAI API

## การใช้งาน
1. ตั้งค่า API keys ในไฟล์ `.env` (GROQ_API_KEY, OPENAI_API_KEY, TOGETHER_API_KEY)
2. สร้างอินสแตนซ์โมเดลจากคลาสที่ต้องการในสคริปต์หลัก
3. เรียกใช้เมธอดการส่งพรอมต์และรับผลลัพธ์ตามเอกสารในคลาสนั้น

## หมายเหตุ
- หากเพิ่มผู้ให้บริการ LLM ใหม่ ให้เพิ่มโมดูลใหม่โดยสืบทอดจาก `LLMBase` และเขียน wrapper สำหรับ API ใหม่
