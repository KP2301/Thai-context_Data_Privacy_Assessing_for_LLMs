# midterm

โฟลเดอร์นี้เก็บไฟล์และกราฟที่ใช้สำหรับการนำเสนอผลการทดลองในช่วงกลางภาคเรียน

## โครงสร้าง
- `final_total_result/` - รายงานสรุปรวมสุดท้าย (ไฟล์ `.txt`)
- `pictures_dea/`, `pictures_ja/` - รูปภาพ/กราฟสรุปผลลัพธ์จาก DEA และ JA
- `token/`, `word/` - ข้อมูลสถิติเกี่ยวกับจำนวนโทเคนและคำที่ใช้
- `token_mid_present.txt`, `word_count_mid_present.txt` - สรุปตัวเลขสำหรับนำเสนอ

## การใช้งาน
- ใช้เพื่อจัดเก็บผลลัพธ์และกราฟสำหรับการนำเสนอ และเพื่อเปรียบเทียบแนวทางต่าง ๆ

## หมายเหตุ
- โฟลเดอร์นี้เป็นพื้นที่เก็บงานนำเสนอ หากต้องการใช้งานทดลองเพิ่มเติม ให้ใช้โฟลเดอร์อื่นที่เหมาะสมและคัดลอกผลลัพธ์มาไว้ในนี้
