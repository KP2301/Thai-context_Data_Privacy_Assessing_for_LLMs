# temp

โฟลเดอร์นี้ใช้เก็บไฟล์ชั่วคราวที่สร้างขึ้นระหว่างการทดลองและการพัฒนา

## ตัวอย่างไฟล์
- `shot_dea.txt` - ตัวอย่างผลลัพธ์หรือข้อมูลชั่วคราวจาก DEA
- `test/` - โฟลเดอร์สำหรับทดสอบโค้ดหรือสคริปต์

## หมายเหตุ
- เนื่องจากเป็นไฟล์ชั่วคราว ไม่ต้องเก็บไว้ใน repository หากไม่จำเป็น
- สามารถลบหรือทำความสะอาดได้ทุกเมื่อหากไม่ต้องการใช้งานต่อ
